# GitHub Fork

- 기존 Repository Fork
- PR : 나의 Repository에서 작업 -> 기존 Repository 브랜치로 pull request
- Pull : 기존 Repository에서 pull 



## 기존 Repository Fork

- 원하는 repo 들어가서 상단의 fork 버튼 클릭
- -> 내 계정에 해당 repo 생김



## 기존 Repository로 Pull Request

- github 나의 repo에서 pull request 클릭
- base branch를 기존 repo의 원하는 branch로…
- pr 날림
